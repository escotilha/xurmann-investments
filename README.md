# Xurmann Investments Website

A professional landing page for Xurmann Investments - a private investment office focused on long-term capital allocation across technology, software, and emerging markets.

## Overview

This is a static single-page website built with:
- Pure HTML5
- Inline CSS with CSS custom properties
- Vanilla JavaScript for animations
- Google Fonts (Sora typeface)

## Deployment

This site is deployed via Cloudflare Pages with automatic deployments from GitHub.

**Live URL:** https://xurman.com

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js (if installed)
npx serve
```

Then visit `http://localhost:8080`
